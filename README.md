# Rhyming-Neural-Net
LSTM network created using Tensorflow and Keras frameworks.
For provided word, the net generates a rhyme with ~80% performance.



Sample action:

INPUT   OUTPUT

tissue	purview

attach	wristwatch

grace	showplace

abate	vacillate

humdrum	aurum

amphibole	pole

electrician	ignition

tomboy	malloy

expressivity	interactivity

canine	serpentine

mausoleum	aurum
